### #27 Validate Status Transitions

**Acceptance Criteria**

Permitido:

```text
PENDING -> PREPARING
PREPARING -> OUT_FOR_DELIVERY
OUT_FOR_DELIVERY -> COMPLETED
```

Bloquear:

```text
PENDING -> COMPLETED
PENDING -> OUT_FOR_DELIVERY
PREPARING -> COMPLETED
```

# Authorization

### #28 Create Role Enum

```text
ADMIN
ATTENDANT
KITCHEN
DELIVERY
GUEST
```

GUEST representa um usuário autenticado sem permissões operacionais. Apenas ADMIN pode promover um GUEST para outra role.
